# Gallery Website

A web-based image gallery with features to:

- Add images to favorites
- Search images
- Apply category filters

## Features

- Responsive image grid
- Favorite button per image
- Real-time search
- Category filtering

## Technologies Used

- HTML
- CSS / Bootstrap
- JavaScript
- Pexels API
- Local Storage
